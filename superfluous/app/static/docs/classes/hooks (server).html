Class hooks (server)

Class defined in: core/server/hooks.js:1

The hooks module is responsible for exposing events in the core that can be overridden.

The main type are 'setup' and 'invocations'. setup is for middleware, generally, and invocations happen for per request events, but the line is blurry.
